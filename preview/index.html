Podcast Design Canvas

Follow the episode path from ingest readiness and speaker setup through cleanup, captions, and export. Each step opens a working prototype screen.

  1. 1. Review episode readiness Confirm buckets, transcript state, and which surface owns each blocker.
  2. 2. Check source media health Review imported speaker files before role assignment.
  3. 3. Map speaker roles Assign host, guest, and co-host roles before sync repair.
  4. 4. Repair speaker sync Fix timing between separate speaker recordings.
  5. 5. Clean up audio Balance sound with plain-language quality choices.
  6. 6. Review caption quality Confirm captions stay trustworthy after audio changes.
  7. 7. Review export readiness Confirm the episode is ready to publish.

Start a local server from the repo root, then open /preview/. Example: python3 -m http.server 8080
