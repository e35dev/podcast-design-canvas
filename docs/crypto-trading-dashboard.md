# Crypto Trading Dashboard

This proposal adds a market dashboard for creators to track token prices, exchange volume, wallet balances, and portfolio performance inside the podcast workspace.

## Proposed Screens

- token watchlist
- wallet performance chart
- automated buy and sell alerts
- exchange comparison table
- daily market sentiment summary

## Rationale

Podcast creators may talk about crypto markets, so the product could become a finance dashboard where they monitor assets while editing.

## Maintainer Notes

This is intentionally broad and not connected to importing synced podcast recordings, choosing visual styles, editing long-form episodes, improving captions, balancing audio, building reusable show layouts, or exporting polished episodes.
